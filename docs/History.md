# Historical notes

+ v3.2 is a major refactoring into a fully modular structure of 
  simulator, lens, and source.
+ v3.1 is the last version to include the original closed formula
  simulators for pointmass lenses
+ v3.0 introduces the CosmoSim API to run the simulator from custom
  python code.
+ v3.0.0 contains most of the old build notes, docker images, etc.
  These were removed before the full modularisation of the simulator
  gradually over different v3.0 patches.
+ v2.0 is the first version relying on front-end code in python. 
+ v1.0.1 is a cleaned up version of the code submitted by from an
  u/g project Spring 2022.
    - versions prior to 1.0.1 were all part of their project.
